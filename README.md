# Dark Future Core
A streamlined Survival Gameplay mod for Cyberpunk 2077 based on the full version of [Dark Future - Urban Survival Gameplay](https://github.com/DarkFortuneTeller/DarkFuture).

## Documentation
Please see the [Dark Future Wiki](https://wiki.darkfuture2077.com).

## Getting Started
### Requirements
Cyberpunk 2077 2.31 (older versions are NOT supported). The mod is Phantom Liberty-aware but the expansion is not required.

[Redscript](https://www.nexusmods.com/cyberpunk2077/mods/1511) 0.5.31+

[Red4Ext](https://www.nexusmods.com/cyberpunk2077/mods/2380) 1.29.1+

[ArchiveXL](https://www.nexusmods.com/cyberpunk2077/mods/4198) 1.26.1+

[TweakXL](https://www.nexusmods.com/cyberpunk2077/mods/4197) 1.11.2+

[Codeware](https://www.nexusmods.com/cyberpunk2077/mods/7780) 1.18.1+

[Mod Settings](https://www.nexusmods.com/cyberpunk2077/mods/4885) 0.2.21+

[Input Loader](https://www.nexusmods.com/cyberpunk2077/mods/4575) 0.2.3+

### Installation
Extract the release archive `DarkFutureCore-x.x-for-x.x.zip` into the Cyberpunk 2077 directory.

Or, install the release archive using Vortex.

Or, install the mod directly from the [Dark Future Core Nexus Mods page](https://www.nexusmods.com/cyberpunk2077/mods/26956).

### Uninstallation
Toggle **Enable Dark Future** to **Off** in the Dark Future Core Mod Settings Menu. Save your game and exit.

Then, remove all Dark Future Core files from the Cyberpunk 2077 directory. Or, uninstall the mod using Vortex.
